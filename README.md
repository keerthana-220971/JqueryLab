# 📘 jQuery Lab

This project demonstrates fundamental concepts of **jQuery** including selectors, attributes, events, DOM traversal, styling, and effects.

---

## 🎯 Objective

The objective of this lab is to understand and implement:

- jQuery Selectors
- Attribute manipulation
- Event handling
- Style manipulation
- DOM Traversing
- jQuery Effects and Animations
- Git version control workflow

## 🧩 Features Implemented

### 🔹 1. jQuery Selectors
- Element, ID, Class selectors
- Multiple element selection
- Basic DOM targeting

---

### 🔹 2. Attribute Methods
- `val()` → Get input values  
- `attr()` → Change attributes  
- `prop()` → Handle boolean properties  

---

### 🔹 3. Event Handling
- Mouse events (`click`, `hover`)
- Keyboard events (`focus`, `blur`)
- Interactive user actions

---

### 🔹 4. Style Manipulation
- `css()` → Change styles
- `addClass()` / `removeClass()`
- `toggleClass()` for dynamic UI

---

### 🔹 5. DOM Traversing
- Parent, children, and sibling navigation
- Methods like `siblings()`, `parent()`, `children()`

---

### 🔹 6. jQuery Effects
- `hide()`, `show()`, `toggle()`
- `fadeIn()`, `fadeOut()`
- `slideDown()`, `slideUp()`

---

### 🔹 7. Animations
- `animate()` for custom animations
- Moving elements, resizing, opacity changes
